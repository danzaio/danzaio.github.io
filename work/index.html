Understand the work

const brief = defineWork({ user, job, sources, risk, decision });

Design the product slice

Shape the smallest useful flow: intake, context retrieval, model call, review surface, action, and handoff.

const slice = designFlow({ intake, retrieval, model, review, action, handoff });

Connect context

Connect context

Build the RAG or source layer from current references, internal documents, browser state, tables, or API data.

const context = buildContext({ documents, browserState, tables, apiData });

Automate the workflow

Automate the workflow

Use a direct function, browser automation, sequential workflow, or orchestrated worker lanes based on risk and dependency shape.

const route = chooseRoute({ task, dependency, risk, approvalNeeded });

Pause risky actions

Pause risky actions

Put human approval before external effects, sensitive data use, irreversible edits, exports, or ambiguous tradeoffs.

await requireApproval({ action, data, risk, rollback });

Make output inspectable

Make output inspectable

Return a product view, table, CSV, spreadsheet, proof packet, or structured artifact instead of a loose transcript.

const artifact = createArtifact({ schema, rows, proof, exportReady });

Validate the system

Validate the system

Check source use, bad inputs, branch behavior, browser state, review notes, and whether the output can be maintained.

await validate({ flow, sources, branches, reviewSurface, handoff });

Ship the handoff

Ship the handoff

Close with what works, what is public-safe to discuss, what remains risky, and what the next engineer or operator should do.

return handoff({ evidence, publicCopy, risks, nextAction });
